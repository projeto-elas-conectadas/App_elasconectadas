# Contexto do Projeto "Elas Conectadas" para Prototipagem

**O que é o projeto:**
O aplicativo "Elas Conectadas" é uma plataforma mobile (com backend funcional em NestJS/MySQL e frontend em Flutter) desenhada primariamente para empoderar e conectar mulheres. O foco é criar uma rede segura de interação focada em negócios, capacitação e networking.

**Quais são os perfis de usuário:**
O sistema possui hoje usuários comuns (USER) e administradoras (ADMIN).
- Conta com um sistema de verificação de conta seguro (conta entra como `UNVERIFIED` e torna-se `VERIFIED` via código OTP - One Time Password, enviado por email).

**Quais são as principais funcionalidades mapeadas até o momento (baseadas no Banco de Dados):**

1. **Gestão de Perfil:**
   - As mulheres cadastram: Nome, Email, Telefone, Data de Nascimento, Ocupação/Profissão, Localização, Mini-bio, e Foto de Perfil.
2. **Sistema de Anúncios / Classificados (Ads):**
   - As usuárias podem explorar e se relacionar com anúncios classificados por tipos: Produtos (`PRODUCT`), Serviços (`SERVICE`) e Colaborações (`COLLAB`). 
   - Os anúncios possuem título, capa, data e conteúdo. O app mapeia quais usuárias estão interessadas ou interagindo com quais anúncios.
3. **Mural de Oportunidades / Eventos (Posts):**
   - Espaço gerenciado pelas administradoras para divulgar iniciativas.
   - Os posts são classificados em: Eventos (`EVENT`) e Cursos (`COURSE`). Têm título, imagem de capa e conteúdo descritivo.

**O Desafio para a Prototipagem UX/UI:**
O código base já existe, mas o foco agora é a Experiência do Usuário (UX) e a Interface (UI). Precisamos desenhar *como* as mulheres vão se sentir dentro do app e qual a jornada delas.

1. **Mapeamento de Jornada:** Como deve ser a tela inicial (Home) para que seja acolhedora e já direcione a mulher para criar negócios, ver cursos ou colaborar?
